# 项目介绍 - AI上下文

## 项目基本信息
- **项目名称**: Admin Template
- **类型**: 后台管理系统模板
- **架构**: 前后端分离
- **主要模块**: `admin-frontend`、`admin-backend`、`admin-mobile`

## 项目特点
- 前后端分离架构，前端 Vue 3 + TypeScript，后端 Spring Boot 4
- 完整的 RBAC 权限体系
- 集成 Element Plus 组件库
- 多端支持：Web 端后台 + App 移动端
- 现代化技术栈，代码结构清晰

## 主要功能
- 用户管理：增删改查、密码重置
- 角色管理：增删改查、权限分配
- 菜单管理：系统菜单配置
- 数据统计：可视化图表
- 日志管理：操作日志、登录日志
- 系统配置：基础配置管理
