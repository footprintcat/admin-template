# 项目介绍

本项目是一个完整的后台管理系统模板，包含前端、后端及 App 移动端三个部分，提供了基础的用户管理、权限控制、数据统计等功能，可以作为各类企业级后台系统的快速开发脚手架。

## 项目特点

- **前后端分离架构**：前端使用 Vue 3 + TypeScript 开发，后端基于 Spring Boot 4 构建，实现了清晰的前后端分离
- **完整的权限体系**：支持基于角色的权限控制（RBAC），可以灵活配置用户权限
- **丰富的 UI 组件**：前端集成了 Element Plus 组件库，提供了丰富的 UI 组件和交互效果
- **多端支持**：包含 Web 端后台管理系统和 App 移动端，满足不同场景的使用需求
- **现代化技术栈**：采用当前主流的技术栈，代码结构清晰，易于维护和扩展

## 主要功能

- 用户管理：用户的增删改查、密码重置等
- 角色管理：角色的增删改查、权限分配等
- 菜单管理：系统菜单的配置和管理
- 数据统计：提供可视化的数据统计图表
- 日志管理：系统操作日志和登录日志的记录和查询
- 系统配置：系统基础配置的管理
